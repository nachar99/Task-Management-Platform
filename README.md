# Task Management Platform

A full-stack MERN application for managing projects and tasks with role-based access control.

Users can create projects, assign tasks to team members, update task status, and organize work efficiently. The platform also includes an admin panel for managing users and system content.

---

## Features

- Secure user authentication using JWT
- Project creation and management
- Task creation, editing, and deletion
- Assign tasks to users via email
- Task status and priority updates
- Role-based access control (User / Admin)
- Admin panel for managing users, projects, and tasks
- Light and dark mode support
- Persistent login sessions
